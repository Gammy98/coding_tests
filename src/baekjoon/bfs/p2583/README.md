# 영역 구하기 (2583)

## 🔍 문제 정보

- **출처:** [https://www.acmicpc.net/problem/2583](https://www.acmicpc.net/problem/2583)
- **알고리즘 태그:** #BFS #너비_우선_탐색

## 💡 접근 방법
사각형마다 방쿤처리를 해주고 나머지공간에서 bfs를 돌려 리스트에 각각의 넓이 구해서 넣는다
그후 개수와 넓이 각각 출력 + collection이용해 오름차순 출력가능

(⭐핵심) 하나의 리스트를 만들어 bfs로 계산한 넓이를 각각 저장시키고 개수는 리스트의 개수로 구한다

## ❗️ 배운 점 / 어려웠던 점
- 가로세로 헷갈리는게 조금씩 잡혀가는 중
- 스트링을 자유롭게 만지고 싶으면 stringbuilder사용
