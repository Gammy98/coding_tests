## 💡 접근 방법

거리를 나타내는 정수배열을 만들어 큐에 넣을때마다 숫자를 하나씩 늘려줘서 계산한다.

(⭐핵심) 초기화를 0으로 해주고 시작점을 1로 해줘서 무한반복에 빠지지 않도록 한다
.

## ❗️ 배운 점 / 어려웠던 점
