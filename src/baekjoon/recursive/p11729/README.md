# 텀 프로젝트 (11729)

## 🔍 문제 정보

- **출처:** [https://www.acmicpc.net/problem/11729](https://www.acmicpc.net/problem/11729)
- **알고리즘 태그:** #DFS #재귀

## 💡 접근 방법

(⭐핵심) n-1 개의 탑이 2번째 장대로 이동시키고 가장 큰 원판을 3번으로 이동 그후 2번에있던 원판들을 3번으로 이동시키면 된다는 
재귀적 관점이 핵심


## ❗️ 배운 점 / 어려웠던 점
- 함수를 자꾸 타고 들어가서 생각하는데 재귀로 풀수있는 문제다 싶으면 base condition과 바로 n일때 알수있는 과정들을 생각하며 귀납적해석이 중요하다.
- 원판 이동은 달라지기 때문에 인자로 받는다는 생각이 어려웠다
